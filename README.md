## How to Use

Clone the project

```bash
    git clone https://github.com/pras306/redux-toolkit-tutorial.git
```

Go to project Directory

```bash
    cd redux-toolkit-tutorial
```

Install Dependencies

```bash
    npm install
```

Start the server

```bash
    npm start
```
